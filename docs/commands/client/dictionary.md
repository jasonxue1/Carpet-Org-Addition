根据对象翻译后的名称查询对应ID

## 语法

- `/dictionary item <item>`：根据翻译后的物品名称查询一个物品的ID
- `/dictionary block <block>`：根据翻译后的方块名称查询一个方块的ID
- `/dictionary entity <entity>`：根据翻译后的实体名称查询一个实体的ID
- `/dictionary enchantment <enchantment>`：根据翻译后的魔咒名称查询一个魔咒的ID
- `/dictionary statusEffect <statusEffect>`：根据翻译后的状态效果名称查询一个状态效果的ID
- `/dictionary biome <biome>`：根据翻译后的生物群系名称查询一个生物群系的ID

## 参数

- 命令参数中，只要翻译后名称中不带有空格字符，字符串就不需要加双引号

## 示例

- 查询“钻石”的物品ID
    - `/dictionary item 钻石`
